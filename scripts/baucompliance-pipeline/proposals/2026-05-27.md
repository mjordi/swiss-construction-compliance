## Proposal S — Connect the protocol wizard photo action to Tech Vault
- **Title:** Connect the protocol wizard photo action to Tech Vault
- **Problem:** In `app/dashboard/page.tsx`, the Step 2 defect section shows a prominent photo button next to defect entry, but it is a dead control with no navigation or capture behavior. That breaks user expectation right where they are most likely to want to attach evidence.
- **Scope:** Turn the existing photo control into a real Tech Vault handoff using the current project name as the Vault search/filter seed, preserve the narrow scope to navigation only, and add focused regression tests that verify the handoff URL and that it stays in sync when the project name changes.
- **Expected impact:** Removes a misleading dead control and gives users a fast path from documenting a defect to finding the relevant evidence workspace.
- **Risk (1-10):** 2
- **Effort/size:** Small

## Proposal M — Add evidence readiness guidance to the protocol wizard
- **Title:** Add evidence readiness guidance to the protocol wizard
- **Problem:** Users can finalize a protocol without any clear reminder of whether supporting evidence already exists for the linked project, which weakens the handoff between protocol creation and later claim defense.
- **Scope:** Surface a lightweight evidence-status panel in the wizard using linked case/project context, show whether matching Vault material exists, and provide explicit follow-up guidance before finalization.
- **Expected impact:** Improves evidence completeness at the moment of protocol creation and reduces later backtracking.
- **Risk (1-10):** 5
- **Effort/size:** Medium

## Proposal L — Build a case-centric evidence and protocol workspace
- **Title:** Build a case-centric evidence and protocol workspace
- **Problem:** Users still have to move across Cases, Protocols, Vault, and Deadlines to understand one claim’s full state. The product lacks a single place where protocol status, evidence, deadlines, and follow-up readiness converge.
- **Scope:** Introduce a dedicated per-case workspace that unifies protocol generation, evidence access, follow-up checklist progress, and deadline context behind one route and navigation model.
- **Expected impact:** Establishes a much clearer operational backbone for managing Swiss construction claims end to end.
- **Risk (1-10):** 8
- **Effort/size:** Large
