## S — Clean invalid dashboard case handoff URLs after lookup

- **Problem:** `/dashboard?case=<id>` is consumed only when the requested case exists. If the case id is stale or unauthorized, the wizard stops pending the handoff but leaves the stale `case=` parameter in the URL, so refreshed/shared links continue to imply a link that was never applied.
- **Scope:** In `app/dashboard/page.tsx`, when cases have loaded and the requested case is not found, remove only the invalid `case` parameter while preserving unrelated query params. Add focused RTL coverage in the existing dashboard linked-case test file.
- **Expected impact:** Makes cross-workflow handoffs safer and reduces confusing stale URLs without changing data persistence or case loading semantics.
- **Risk (1-10):** 2
- **Effort/size:** Small

## M — Add a dedicated stale-linked-case recovery banner for URL handoffs

- **Problem:** Users arriving from case/vault handoffs with a stale `case` id receive no explicit explanation that the handoff failed; the wizard simply falls back to standalone protocol creation.
- **Scope:** Add localized copy and a compact dismissible warning when a URL case handoff is not found, with tests for preservation of valid sibling params and normal standalone creation.
- **Expected impact:** Improves trust and troubleshooting around cross-workflow navigation, especially after archived/deleted cases.
- **Risk (1-10):** 4
- **Effort/size:** Medium

## L — Unified dashboard handoff state contract across Cases, Vault, and Protocol Wizard

- **Problem:** Cross-workflow handoffs are implemented as local URL/query-state conventions in each surface, making it easy for stale params, filters, and local drafts to diverge.
- **Scope:** Define a shared handoff helper module, normalize/sanitize query params across dashboard/cases/vault, and add integration coverage for multi-step handoff flows.
- **Expected impact:** Strategic reliability improvement for the product's workflow graph and future deep-linking work.
- **Risk (1-10):** 7
- **Effort/size:** Large / strategic
