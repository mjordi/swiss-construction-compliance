## Small (S): Mark reminder preset toggles with pressed state

- **Problem:** Reminder preset buttons visually indicate selection, but assistive technologies do not receive the same selected/unselected state. This makes public/dashboard calendar reminder controls less understandable for keyboard and screen-reader users.
- **Scope:** Add `aria-pressed` to reminder preset buttons in the dashboard deadlines page and the public Rügefrist calculator, with focused tests asserting default, toggled, and URL-hydrated pressed states.
- **Expected impact:** Improves accessibility of a recently shipped user-facing reminder feature with minimal behavioral risk.
- **Risk:** 2/10
- **Effort/size:** S

## Large (L): Add a consolidated deadline command center

- **Problem:** Users currently manage deadlines through separate calculator, dashboard deadline, case list, and vault surfaces. There is no single cross-project calendar view that combines legal deadlines, reminder exports, checklist status, and project context.
- **Scope:** Design and implement a new dashboard command center that aggregates case deadlines, supports filtering by status/regime/project, exposes calendar exports, and links directly into case/vault workflows.
- **Expected impact:** Strong strategic value for compliance teams managing many projects, turning the app from calculators into an operational deadline workspace.
- **Risk:** 8/10
