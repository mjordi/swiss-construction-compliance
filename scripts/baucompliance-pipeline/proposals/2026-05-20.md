## Proposal S — Add explicit failure feedback for case create/delete actions
- **Title**: Add explicit failure feedback for case create/delete actions
- **Problem**: Creating or deleting a case currently awaits Supabase calls without surfacing failure state, so users can believe a case saved or disappeared even when the mutation failed.
- **Scope**: Add localized create/delete error feedback to `/dashboard/cases`, preserve the add-case form state on failed creates, and add focused tests for mutation failure handling.
- **Expected impact**: Improves reliability and trust on one core workflow with low implementation risk.
- **Risk (1-10)**: 3
- **Effort/size**: Small

## Proposal M — Add a direct “create protocol” handoff from cases into the dashboard wizard
- **Title**: Add a direct “create protocol” handoff from cases into the dashboard wizard
- **Problem**: Users can identify urgent or protocol-ready cases in `/dashboard/cases`, but they still have to navigate manually to `/dashboard` and reselect the same case before they can generate a linked handover protocol.
- **Scope**: Add a per-case CTA from `/dashboard/cases` into `/dashboard`, support hydrating a requested linked case from a URL param, preserve existing stale/missing-case safeguards, and add focused regression coverage for the new handoff.
- **Expected impact**: Creates a high-value bridge from legal triage into document generation, reducing friction on one of the product’s most important workflows.
- **Risk (1-10)**: 4
- **Effort/size**: Medium

## Proposal L — Build a unified case workspace across cases, protocols, and vault evidence
- **Title**: Build a unified case workspace across cases, protocols, and vault evidence
- **Problem**: Users still have to coordinate deadlines, protocol generation, and evidence review across separate pages when handling one dispute or handover issue.
- **Scope**: Introduce a shared case workspace with protocol activity, evidence context, next-action prompts, and cross-workflow status rollups in one place.
- **Expected impact**: Meaningfully improves end-to-end product coherence for Swiss construction claims handling.
- **Risk (1-10)**: 7
- **Effort/size**: Large
