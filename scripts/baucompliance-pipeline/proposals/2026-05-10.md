## Proposal S — Surface clipboard failure feedback on deadline share links
- **Title**: Surface clipboard failure feedback on deadline share links
- **Problem**: The deadlines calculator share-link action currently falls back to doing nothing when the clipboard write fails, so users cannot tell whether the link copied or whether they need to retry with different browser permissions.
- **Scope**: Add localized failure feedback to the deadlines share-link button state, keep the existing copied-success feedback, and add focused regression tests plus locale-key coverage for the new feedback path.
- **Expected impact**: Makes a core share/export flow more trustworthy without expanding beyond one page, a few locale strings, and targeted tests.
- **Risk (1-10)**: 2
- **Effort/size**: Small

## Proposal L — Build a unified evidence-and-deadlines case workspace
- **Title**: Build a unified evidence-and-deadlines case workspace
- **Problem**: Users still have to piece together deadlines, documents, and protocol evidence across multiple dashboard surfaces when managing one dispute or handover issue.

- **Effort/size**: Large
