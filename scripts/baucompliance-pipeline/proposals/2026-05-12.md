## Proposal S — Harden cases share-link feedback against failed or stale clipboard writes
- **Title**: Harden cases share-link feedback against failed or stale clipboard writes
- **Problem**: The cases dashboard share-link action still fails silently on clipboard rejection and can surface outdated copied feedback if an earlier async clipboard write resolves after the user has already changed the filtered view.
- **Scope**: Add localized error feedback and request/version guarding to the cases share-link button state, clear stale feedback when the shared view changes, and add focused regression tests plus locale-key coverage.
- **Expected impact**: Improves trust in a core compliance triage workflow while staying inside one page, one test file, and locale updates.
- **Risk (1-10)**: 3
- **Effort/size**: Small
