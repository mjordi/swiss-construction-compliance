The best daily slice is another small reliability fix on a core workflow rather than a larger rewrite: the dashboard handover wizard already persists drafts, but its discard/new-flow behavior can silently recreate an empty saved draft after the user asks to remove it.

## S — Make dashboard draft discard truly clear persisted state

- **Problem:** The dashboard wizard `clearDraft()` removes `baucompliance:wizard-project-draft`, then immediately resets React state. The draft persistence effect observes those resets and writes a fresh empty draft with a new timestamp back to localStorage, so discard/new does not actually leave storage clean.
- **Scope:** Add a narrow one-shot suppression for the next draft persistence cycle after explicit discard/new, and add a focused regression test around the visible discard action.
- **Expected impact:** Users who discard a handover draft or start a new protocol get a genuinely clean wizard state, avoiding confusing stale draft metadata and reducing surprise on the next visit.
- **Risk:** 3/10 — localized to dashboard draft persistence; testable with existing dashboard mocks.
- **Effort/size:** S

## M — Add a handover draft recovery panel with explicit resume/discard choices

- **Problem:** The wizard auto-restores draft content but does not give a strong up-front recovery choice when returning after time away.
- **Scope:** Add a compact recovery panel on step 1 with last-saved timestamp, resume copy, and discard action; update translations and interaction tests.
- **Expected impact:** Better trust for field teams resuming partially completed handover protocols.
- **Risk:** 5/10 — touches visible UX, i18n, and draft state interactions.
- **Effort/size:** M

## L — Offline-ready handover drafting foundation

- **Scope:** Design and phase in an offline queue for protocols/photos with sync status, conflict handling, and recovery UI.
- **Expected impact:** Strategic field reliability improvement for the core product workflow.
- **Risk:** 8/10 — broad architecture and data consistency impact.
- **Effort/size:** L
