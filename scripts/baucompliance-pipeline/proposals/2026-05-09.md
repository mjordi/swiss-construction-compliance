# Research Proposals — 2026-05-09

## PM Insights

- The cases dashboard is now central to deadline follow-up, but its checklist interactions still fail silently if persistence breaks. That creates compliance-risky false confidence on evidence, notice, and reminder tracking.

## Proposal S — Safeguard follow-up checklist persistence on the cases dashboard
- **Title**: Safeguard follow-up checklist persistence on the cases dashboard
- **Problem**: Case follow-up checklist toggles update the UI optimistically but do not recover or notify users when the Supabase write fails. That can leave teams believing a compliance step was recorded when it was not.
- **Scope**: Normalize checklist state, roll back failed checklist toggles, show localized inline save feedback on the affected case card, and add focused regression tests for the helper logic.
- **Expected impact**: Reduces silent data-loss risk in a core follow-up workflow without expanding beyond one dashboard surface plus translations/tests.
- **Risk (1-10)**: 3
- **Effort/size**: Small

## Proposal M — Add one-click protocol drafting from case cards
- **Title**: Add one-click protocol drafting from case cards
- **Problem**: Case cards expose deadline and checklist context, but users still need to navigate separately into the handover wizard and manually reconnect the relevant case.
- **Scope**: Add a case-card CTA that deep-links into the dashboard protocol wizard with the selected case preloaded and verified against the existing draft-handling flow.
- **Expected impact**: Shortens the path from defect follow-up to compliant protocol drafting in a high-value workflow.
- **Risk (1-10)**: 5
- **Effort/size**: Medium

## Proposal L — Add a dedicated case detail workspace with protocol and evidence context
- **Title**: Add a dedicated case detail workspace with protocol and evidence context
- **Problem**: Users still have to move between cases, protocols, and the vault to assemble the full legal and evidence story for one project.
- **Scope**: Introduce a richer case-detail surface that combines deadlines, follow-up progress, linked protocols, and evidence context into one workspace.
- **Expected impact**: Deepens product usefulness for active claims handling and reduces cross-page context switching.
- **Risk (1-10)**: 7
- **Effort/size**: Large
