## Proposal S — Localize the Technical Vault shell and empty states
- **Title**: Localize the Technical Vault shell and empty states
- **Problem**: The Vault page still presents its title, search placeholder, loading/error states, empty-state guidance, status chips, and creation CTAs in hardcoded English, which breaks the multilingual promise for German/French/Italian users in a legally sensitive evidence workflow.
- **Scope**: Move the Vault shell strings and empty-state copy onto translation keys, update the Vault helper to return translation-key metadata instead of raw English strings, and add focused regression tests for the helper/locales coverage.
- **Expected impact**: Makes the evidence workspace feel consistent with the rest of the product for non-English users and reduces comprehension friction when searching or triaging stored project evidence.
- **Risk (1-10)**: 2
- **Effort/size**: Small
