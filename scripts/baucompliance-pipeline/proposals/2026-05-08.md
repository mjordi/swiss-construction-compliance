## Proposal S — Add a copy-link action for filtered case triage views
- **Title**: Add a copy-link action for filtered case triage views
- **Problem**: The cases dashboard already serializes filter/search state into the URL, but users have no obvious affordance to copy the current triage view and share it with collaborators. This hides a useful workflow behind manual address-bar copying.
- **Scope**: Add a small share-link button in the cases filter toolbar, copy the current filter/search URL to the clipboard, show localized copied-state feedback, reset stale copied feedback when the view changes, and add focused regression tests.
- **Expected impact**: Makes case triage views explicitly shareable for teams without changing the underlying data model or expanding beyond one page plus translations/tests.
- **Risk (1-10)**: 3
- **Effort/size**: Small

## Proposal M — Add one-click protocol drafting from case cards
- **Title**: Add one-click protocol drafting from case cards
- **Problem**: Case cards show checklist/protocol context, but users still need to navigate separately into the handover wizard and manually select the related case before drafting a protocol.
- **Scope**: Add a case-card CTA that deep-links into the dashboard protocol wizard with the case preselected, preserve project context, and add focused tests for the linked-case handoff.
- **Expected impact**: Reduces workflow friction between active defect follow-up and protocol creation in a core product journey.
- **Risk (1-10)**: 5
- **Effort/size**: Medium

## Proposal L — Add a dedicated case detail workspace with protocol and evidence context
- **Title**: Add a dedicated case detail workspace with protocol and evidence context
- **Problem**: Users still need to move between the cases dashboard, protocol flow, and vault to understand the full legal and evidence context for one case.
- **Scope**: Add a richer case detail surface that combines deadlines, checklist state, linked protocols, and evidence context into one workspace.
- **Expected impact**: Deepens product usefulness for active compliance follow-up and reduces cross-page context switching.
- **Risk (1-10)**: 7
- **Effort/size**: Large
