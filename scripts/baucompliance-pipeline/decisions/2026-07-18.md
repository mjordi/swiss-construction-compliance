This is not just internal cleanup: checklist readiness is surfaced in case scan cards, Vault audit snapshots, and future activity summaries, so writing a complete normalized payload prevents hidden legal-readiness drift.

Update focused linked-case finalization tests in `__tests__/dashboard-linked-case-retry.test.tsx` to assert complete normalized payloads for null and sparse persisted checklist data. Run `npm run test -- __tests__/dashboard-linked-case-retry.test.tsx`, `npm run test`, `npm run lint`, and `npm run build`.
